İş hayatımda yaptığım çalışmalar , projeler ve bulunduğum insan ilişkilerinin deneyim kazanmamda büyük katkısı oldu. Bu süreçte kazandığım deneyimlerin, meslekteki gelişimime büyük katkısı olduğuna inanıyorum. Araştırmayı, yeni teknolojiler hakkında bilgi edinmeyi, gerektiği takdirde onları kullanmayı isterim. Şuana kadar entegrasyon, web yazılım, web servis gibi projeler geliştirdim ve geliştirme sürecinde bulundum. Son çalışmalarım da ağırlıklı olarak .Net core ile Rest Api ve backend projeleri üzerinedir.
